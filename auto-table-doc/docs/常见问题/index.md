## 快速排查

### 表没有创建？

1. 检查是否添加了 `@EnableAutoTable` 注解
2. 检查实体是否添加了 `@AutoTable` 注解
3. 检查包扫描路径是否正确
4. 查看启动日志是否有错误信息

### 字段没有更新？

1. 确认运行模式是 `update`（默认）
2. 检查字段是否被 `@Ignore` 标记
3. 检查字段修饰符是否为 `static` 或 `final`

### 报错 Invalid value type？

通常是类型映射问题，检查：
1. 是否使用了不支持的 Java 类型
2. 是否需要添加 `@ColumnType` 指定数据库类型
