# 安装

## Maven 依赖

> 最新版本请查看 [Maven Central](https://central.sonatype.com/artifact/org.dromara.autotable/auto-table)

::: code-group

```xml [Spring Boot]
<!-- 兼容 Spring Boot 2.x 与 3.x -->
<dependency>
    <groupId>org.dromara.autotable</groupId>
    <artifactId>auto-table-spring-boot-starter</artifactId>
    <version>2.5.10</version>
</dependency>
```

```xml [Solon]
<dependency>
    <groupId>org.dromara.autotable</groupId>
    <artifactId>auto-table-solon-plugin</artifactId>
    <version>2.5.10</version>
</dependency>
```

```xml [纯 Java]
<!-- 包含所有数据库策略 -->
<dependency>
    <groupId>org.dromara.autotable</groupId>
    <artifactId>auto-table-strategy-all</artifactId>
    <version>2.5.10</version>
</dependency>
```

:::

## 按需引入数据库策略

如果只需要特定数据库支持，可以按需引入：

```xml
<!-- MySQL -->
<dependency>
    <groupId>org.dromara.autotable</groupId>
    <artifactId>auto-table-strategy-mysql</artifactId>
    <version>2.5.10</version>
</dependency>

<!-- PostgreSQL -->
<dependency>
    <groupId>org.dromara.autotable</groupId>
    <artifactId>auto-table-strategy-pgsql</artifactId>
    <version>2.5.10</version>
</dependency>

<!-- Oracle -->
<dependency>
    <groupId>org.dromara.autotable</groupId>
    <artifactId>auto-table-strategy-oracle</artifactId>
    <version>2.5.10</version>
</dependency>

<!-- 更多策略... -->
```

## Gradle 依赖

::: code-group

```groovy [Spring Boot]
implementation 'org.dromara.autotable:auto-table-spring-boot-starter:2.5.10'
```

```groovy [Solon]
implementation 'org.dromara.autotable:auto-table-solon-plugin:2.5.10'
```

```groovy [纯 Java]
implementation 'org.dromara.autotable:auto-table-strategy-all:2.5.10'
```

:::

## 版本要求

| 依赖 | 最低版本 |
|------|----------|
| JDK | 1.8+ |
| Spring Boot | 2.x / 3.x |
| Solon | 3.x |
| Mybatis | 3.5+ |
